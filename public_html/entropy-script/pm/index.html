Entropy Script Particle Manager

Entropy Script Particle Manager manages particles (packages) in your EntropyScript project.

GitHub | NPM package
Download using NPM (bundled with entropy script): 

        npm install entropy-script -g
and then use with 

        espm -


Commands

install


                espm install
If there are no other arguments, looks for a particle.json file, and then downloaded particles specified in that file.
Can take another argument, a single particle name. Installs that particle and adds it to the particle.json.

publish


                espm publish
Publishes the particle so that others can use it. Requires a main.es file as an entry point.
The particle will nbe published under the name specified in the particle.json.
